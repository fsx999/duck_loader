Welcome to duck_loader's documentation!
=======================================

Il s'agit d'un chargeur pour gérer les modules de la suite duck

Le module est chargée de deux services :

- Le chargement automatique des Web Apis des modules
- La création d'un dashboard pour un backoffice, la création des points d'entrées des modules dans celui-ci, et
le chargement des urls pour le backoffice des modules.
